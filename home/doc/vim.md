# Visual
```
+v 		        - вся строка целиком
Ctrl+V  		- прямоугольник, часть текста.
```
---
# Move
```
Ctrl+f 			- на стр. вниз
Ctrl+b 			- на стр. вверх
Ctrl+d 			- на пол стр. вниз
Ctrl+u 			- на пол стр. вверх

0 				- в начало текущей строки
$ 				- в конец текущей строки

w 				- слово в право
b 				- слово в лево

W 				- до пробела в право
B 				- до пробела в лево

} 				- абзац вниз
{ 				- абзац вверх

<number>G 		- перейти на конкретную строку

/<text><CR> 	- перейти к тексту
?<text><CR> 	- анал но искать вверх

n 				- повторить поиск
N 				- повторить поиск назад

[[ 				- в начало функции
```
---
# change_mod_INSERT 
```
a 				- после курсора
I 				- переместить в начало строки

s 				- удаляет указанное число символов
S 				- улаляет указанное число строк
de 				- удаляет слово (начиная с курсора)

r<symb> 		- заменяет символ под курсором
<count>r<symb> 	- заменяет несколько символов

<count>x 		- удалить символ(ы) пол курсором
X 				- удалить символ перед курсором
```

# Copy
```
yiw 			- Слово 
yw 				- От курсора до конца слова 
y)   			- От курсора до конца абзаца
y$				- От курсора до конца строки
```


# More 
```
Ctrl-r 			- повторить отмененное действие
sav <name> 		- Сохранить как
e! 				- Отменить все изменения в текущей сессии
r <file>  		- Вставить содержимое из файла
<N>r <file>     - Вставить содержимое из файла с N строки
sh 				- Свернуть vim и перейти в командную оболочку
<N2>,<N2>w file - Создвть файл и скопировать в него содержимое текущего файла

<N2>,<N2>w>> file - Добавить в файл с N1 по N2 строки из текущего файла

```
