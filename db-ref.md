
# Data types #
---

| type     | description                                      |
| -------- | ------------------------------------------------ |
| null     | указывает на отсутствие значения                 |
| integer  | целое значение (положительное или отрицательное) |
| real     | число с плавающей точкой                         |
| text     | строка                                           |
| blob     | бинарные данные                                  |
| NUMERIC  | все 5 типов (любой) ((анал. affinity))           |
